### Installation

- Just, Clone this Repository -
  ```
  $ apt update
  $ apt upgrade
  $ apt install git
  $ apt install nodejs
  $ apt install python
  $ python3 -m pip install -upgrade pip
  $ pip install requests
  $ pip install mechanize
  $ pip install bs4
  $ pip install npm
  $ pip install future
  $ git clone https://github.com/somirit/clonerit.git
  ```

- Now go to cloned directory and run `clonerit.py` -
  ```
  $ cd clonerit
  $ python clonerit.py
  ```
